# Required pubspec.yaml Dependencies

Add these to your `pubspec.yaml` under `dependencies:`:

```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_bloc: ^8.1.3
  equatable: ^2.0.5
  http: ^1.1.0
  intl: ^0.18.1
  flutter_secure_storage: ^9.2.2
  flutter_dotenv: ^5.2.1
  hive: ^2.2.3
  hive_flutter: ^1.1.0
```

Run `flutter pub get` after adding these dependencies.
